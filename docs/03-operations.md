## Day-to-day Workflow
1. Start in [Workbench](05-workbench.md).
2. Plan a change and backup first.
3. Implement the smallest safe change.
4. Validate with explicit tests.
5. Log outcomes in [Change Log](operations/changelog.md).
6. Run [Publish Checklist](operations/publish-checklist.md) before sharing.
